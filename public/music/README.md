# Cinematic tour music

Drop royalty-free tracks here to enable background music in the Cinematic Tour export.

Expected filenames (referenced in src/lib/cinematic/styles.ts MUSIC_TRACKS):
- upbeat.mp3   — Fast Reel style
- ambient.mp3  — Cinematic Slow style
- chill.mp3    — Showcase style

Rename/extend MUSIC_TRACKS to match your files. Without these, "None" is used (silent video).
